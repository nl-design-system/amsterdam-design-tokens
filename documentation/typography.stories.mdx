# Typography

> **Let op!** Dit font wordt niet door de gemeente beschikbaar gesteld en dient voor het ontwerp aangeschaft te worden. Schaf het juiste font aan via: [Avenir® Next Basic Family Pack op Fonts.com](https://www.fonts.com/font/linotype/avenir-next/basic-family-pack). Avenir® Next Basic Family Pack wijkt af van het standaard Avenir font dat op Mac wordt aangeboden.

> Externe designers kunnen de aanschaf van het font doorbelasten aan de opdrachtgever. Designers in dienst van gemeente Amsterdam kunnen een mail sturen naar het [Design System Team](mailto:designsystem@amsterdam.nl).

Bron: [Typography - Amsterdam Design System](https://designsystem.amsterdam.nl/7awj1hc9f/p/632a76-typography)
